# Альп-Пром Бот

Telegram бот для компании «Альп-Пром» (промышленный альпинизм, Харьков)

## Функции

- 📝 Приём заявок
- ❓ FAQ
- 📞 Контакты
- ℹ️ О компании

## Деплой на Render

1. Создай аккаунт на render.com
2. Подключи GitHub
3. New → Web Service
4. Выбери репозиторий
5. Добавь переменные:
   - TELEGRAM_TOKEN
   - ADMIN_ID
6. Деплой!
